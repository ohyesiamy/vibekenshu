## エグゼクティブサマリー

コナミグループ様は2026年3月期Q3で連結売上収益3,530億円（前年同期比+13.6%）、営業利益1,017億円（同+17.4%）と**2期連続で過去最高業績を更新**されています。デジタルエンタテインメント事業が売上の75%（2,661億円）を占め、eFootball、遊戯王、桃太郎電鉄、プロスピ等のIPが牽引しています。

一方で、コナミクリエイティブフロント東京ベイ（有明）の稼働開始、銀座本社でのeスポーツ学院運営、新規タイトル開発の加速など、**開発リソースの需要が急拡大**しています。連結従業員9,268名のうちKDE単体で2,349名のエンジニア・クリエイターが在籍しますが、ゲーム業界全体のエンジニア採用難は深刻です。

**バイブコーディング研修**は、既存開発人材のAI活用能力を飛躍的に向上させ、プロトタイプ開発速度を最大5倍に加速するソリューションです。新規採用に頼らず、**現有戦力の生産性を劇的に引き上げる**ことで、コナミ様の成長戦略を開発面から支えます。

## 2. パッケージ構成

### パッケージA: 即効型 —「Vibe Sprint」
**対象:** デジタルエンタテインメント事業部 プログラマー・プランナー（20名）

> **コンセプト:** 既存開発チームがバイブコーディングで即座に開発速度を向上。2日間で即戦力化。

| 項目 | 内容 |
|---|---|
| **形式** | 対面集中研修（銀座本社 or 有明新拠点） |
| **期間** | 2日間（計16時間） |
| **定員** | 20名/回 |
| **対象者** | ゲームプログラマー、テクニカルプランナー、ツールエンジニア |

#### カリキュラム

**Day 1: バイブコーディング基礎 & ゲーム開発適用（8時間）**

| 時間 | 内容 |
|---|---|
| 09:00-10:30 | バイブコーディング概論：AI駆動開発の原理と業界動向 |
| 10:30-12:00 | ハンズオン①：Claude Code / Cursor を使った高速プロトタイピング |
| 13:00-14:30 | ハンズオン②：ゲームUI/UXプロトタイプをAIで30分で構築 |
| 14:30-16:00 | ハンズオン③：API連携・バックエンドの高速実装 |
| 16:00-17:00 | ケーススタディ：ゲーム業界のAI活用事例（開発工数80%削減の実例） |

**Day 2: 実践 & コナミ様のユースケース適用（8時間）**

| 時間 | 内容 |
|---|---|
| 09:00-10:30 | 運用型ゲーム向けツール開発：ダッシュボード/分析ツールをAIで構築 |
| 10:30-12:00 | チーム演習：コナミIPを題材としたプロトタイプ開発競争 |
| 13:00-15:00 | 実業務適用ワークショップ：各チームの実課題をバイブコーディングで解決 |
| 15:00-16:30 | AI活用ガイドライン策定：品質管理・セキュリティ・著作権配慮 |
| 16:30-17:00 | 成果発表 & 今後のアクションプラン |

| 項目 | 金額（税別） |
|---|---|

### パッケージB（60名）でのROI試算

| 項目 | 金額 |
|---|---|
| 投資額 | ¥18,100,000 |
| 60名の月間人件費 | ¥40,000,000 |

コナミ様の2026年3月期通期営業利益予想1,230億円に対し、パッケージBの投資額1,810万円は**営業利益の0.0015%**。60名のエンジニアの生産性が3倍になることで得られる開発加速効果は、新作タイトルの市場投入時期の前倒しや、運用型ゲームの改善サイクル高速化として、売上収益に直接的にプラス貢献します。
